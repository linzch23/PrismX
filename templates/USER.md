# User Profile
- 用户使用中文交流。
- 用户倾向于让助手首先自我介绍，以了解其能力和行为方式。
- 用户对多代理协作（子代理创建）功能表现出兴趣。
- 用户会询问助手底层的具体模型（如“你是什么模型”），期望获得透明诚实的回答。
- 用户会对工作区内容感兴趣（如要求展示目录文件、工具列表等）。
- 用户曾尝试通过助手获取实时人物动态信息（中山大学校长去向），展示了对外部信息搜索的信任和依赖。
- 用户对助手搜索能力的底层机制有探究兴趣，追问了搜索引擎的具体实现方式（无内置搜索引擎，仅通过 web_fetch 抓取指定 URL）。
- 用户主动提供具体网页链接（微信公众号文章）要求抓取，显示出对助手网络访问能力的具体使用意图。
- 用户尝试使用斜杠命令（如“/tudos”），可能期望助手支持快捷指令，或对该交互模式有使用习惯，需注意后续引导。
